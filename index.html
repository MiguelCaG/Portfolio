Ace Voyage

Este juego de casillas y cartas en tiempo real fue desarrollado en Unity para la asignatura Juegos para Web y Redes Sociales como una práctica de simulación de un entorno de estudio indie, abarcando todas las etapas del desarrollo de un videojuego: desde la elaboración del documento de diseño (GDD) hasta su publicación, incluyendo el estudio de mercado y la prueba de versiones demo.

Proyectos Andorid Studio

Estos juegos fueron desarrollados en Android Studio para la asignatura Desarrollo de Aplicaciones para Dispositivos Móviles. El primero es un juego de preguntas tipo quiz y el segundo es una aplicación con dos juegos: un 2048 y uno tipo wordle.

Bullet Escape

Este juego cuyo objetivo es alcanzar el final de un nivel mientras se esquivan obstáculos fue desarrollado como proyecto para la asignatura de Ingeniería de Videojuegos. En él se implementaron distintos patrones de diseño aprendidos como "Object Pool" y "State".

Question Mark

Este juego de preguntas basado en el juego Reigns fue desarrollado para la asignatura Proceso de Desarrollo de Videojuegos como parte de un proyecto en el que aprender a desarrollar cualquier tipo de software organizándonos siguiendo las metodologías Scrum.
